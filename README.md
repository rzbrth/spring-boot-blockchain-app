## Prerequisites

* Jdk 1.8
* Maven 3.x
* Docker 19.03.x
* Docker Compose 1.25.x

## Installation

```sh
cd first-network
```

```sh
./byfn.sh up -a -s couchdb
```

```sh
cd ..
```

```sh
sudo chmod +x build.sh
```

```sh
./build.sh
```

```sh
docker-compose up -d
```
